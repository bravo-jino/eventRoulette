# Roulette Event Page

Static roulette event page with an admin screen.

## Structure
- `index.html`
- `admin.html`
- `css/style.css`
- `js/app.js`
- `js/admin.js`
- `js/storage.js`

## Run
Open `index.html` or `admin.html` in a browser.

## Deploy
This repository deploys to GitHub Pages from `main` using GitHub Actions.

## Shared Database
By default, the app stores settings and results in each browser's IndexedDB.
To share data across devices, create these tables in Supabase and fill
`js/remote-config.js` with your project URL and anon key.

```sql
create table if not exists roulette_config (
  id text primary key,
  data jsonb not null,
  updated_at timestamptz not null default now()
);

create table if not exists roulette_logs (
  id bigint generated by default as identity primary key,
  timestamp timestamptz not null default now(),
  label text not null
);

alter table roulette_config enable row level security;
alter table roulette_logs enable row level security;

create policy "public read config"
on roulette_config for select
using (true);

create policy "public write config"
on roulette_config for insert
with check (true);

create policy "public update config"
on roulette_config for update
using (true)
with check (true);

create policy "public read logs"
on roulette_logs for select
using (true);

create policy "public insert logs"
on roulette_logs for insert
with check (true);

create policy "public delete logs"
on roulette_logs for delete
using (true);
```

The anon key is public by design. Do not put service-role keys in this repo.
